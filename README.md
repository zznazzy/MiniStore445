# MiniStore445

MiniStore445 is a small **ASP.NET Framework Web Forms** mini online store for **CSE 445/598 Assignments 5 and 6**. 

## Team

- **Bella Rayner**
- **Adrian Simon**

## Project Idea

This project is a **mini online store**. Users can browse sample products and test application components through the GUI.

## How to Open

1. Open `MiniStore445.slnx` in **Visual Studio**.
2. Set **MiniStoreWeb** as the startup project.
3. Build the solution.
4. Run the web app.
5. Open `Default.aspx`.
